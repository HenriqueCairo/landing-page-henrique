# Projeto Prático Web MVC - Henrique Cairo

## Estrutura MVC

Este projeto foi organizado conforme a proposta do professor:

- `model/dados.js`: guarda os dados do perfil e dos projetos.
- `view/index.html`: estrutura visual da página com HTML semântico.
- `controller/app.js`: controla a renderização dos dados e as interações.
- `assets/css/style.css`: estilos, responsividade, flexbox/grid e media queries.
- `assets/image/`: pasta para imagens do projeto.

## Como abrir

Abra o arquivo:

`view/index.html`
